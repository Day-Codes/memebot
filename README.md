# memebot
A bot that plays memes in your Discord channel

## Requirements:
* node LTS
* npm
* git
* [memebot-api server](https://github.com/arjunchib/memebot-api)

## Installation
install requirements first 

```sh
git clone https://github.com/arjunchib/memebot.git
cd memebot
npm install
```

## Usage

Run ``npm run develop``
